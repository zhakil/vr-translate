# Utils Module Context
## 模块职责
提供VR应用中通用的工具函数和辅助类，包括数学计算、数据结构、调试工具和扩展方法，以提高代码复用性和开发效率。
## 开发Context
### 核心功能
1. **数学与几何工具**
   - Vector、Quaternion、Matrix的扩展函数
   - 几何计算（如点线面关系、相交检测）
2. **数据结构**
   - 可序列化的字典或哈希表
   - 对象池，用于复用游戏对象，减少GC开销
3. **调试工具**
   - 在VR环境中实时显示日志或变量值
   - 绘制调试几何体（如射线、包围盒）
4. **C#扩展方法**
   - 简化常用操作的扩展方法集
